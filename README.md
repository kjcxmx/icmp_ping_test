Usage of G:\golang\github\icmp_ping_test\main.exe:
  -c string
        request arp packet count, i.e.:
        -c 4 (default "4")

        ip addr of range, i.e.:
        -i 127.0.0.1/24
  -t string
        host or ip addr, i.e.:
        -t 127.0.0.1 or -t www.baidu.com
